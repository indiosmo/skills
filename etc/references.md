### agent-browser

https://github.com/vercel-labs/agent-browser

agent-browser is a CLI tool that provides individual browser commands. It's designed to be called by AI agents (like Claude Code, Cursor) rather than being an agent itself. The AI decides what commands to run.

*Use Case*: "Give Claude/GPT tools to interact with browsers step-by-step"
